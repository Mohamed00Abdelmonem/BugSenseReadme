# 🐞 BugSense – AI JavaScript Error Explainer

BugSense is a Chrome Extension that captures JavaScript runtime errors on any webpage and explains them in simple, beginner-friendly language using AI.

## 🚀 Features

- 🤖 AI-powered error explanations
- 🔍 Automatic error detection (console, runtime, network)
- 🧠 Understand the cause of errors instantly
- 🛠️ Suggested fixes for each error
- 💻 Simple code examples
- 🌙 Clean UI (Dark / Light mode)
- 🔒 Privacy-friendly (no tracking, no personal data collection)

---

## 🎯 Why BugSense?

Instead of wasting time searching for errors on Google or StackOverflow, BugSense gives you:

- Clear explanation
- Direct fix
- Example code

All inside your browser.

## 🛠️ How it works

1. Captures JavaScript errors from the page
2. Sends error details to AI (Gemini)
3. Displays a simple explanation in UI

---

## 🔐 Privacy

- No personal data is collected
- No tracking
- All data stays within your browser session
