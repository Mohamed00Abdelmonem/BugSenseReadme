# Privacy Policy – BugSense

BugSense respects your privacy.

The extension only reads JavaScript errors and does not track or store user data
## 🔒 Data Collection

BugSense does NOT collect, store, or share any personal data.

We do NOT collect:
- Name
- Email
- Passwords
- Personal information
- Browsing history

---

## 📍 What data is used?

The extension may process:

- JavaScript error messages
- Stack traces
- Page URLs (for debugging context only)

This data is used ONLY to generate explanations.

---

## 🤖 Third-party services

BugSense may send error messages to an AI service (e.g. Google Gemini API) to generate explanations.

- No personal data is included
- Only technical error data is sent

---

## 🔐 Data Storage

- User preferences (like theme or language) are stored locally in the browser
- No data is stored on external servers

---

## 🚫 Data Selling

We do NOT sell, share, or transfer any user data to third parties.

---

## 🛡️ Security

We follow best practices to ensure user safety and privacy.
